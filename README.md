# Catering Search Platform

## Tech Stack

Frontend:
- Next.js
- Tailwind CSS

Backend:
- Node.js
- Express

## Features

- View caterers
- Search by name
- Filter by price
- REST API
- Responsive UI

## Setup

### Backend

cd backend

npm install

npm run dev

### Frontend

cd frontend

npm install

npm run dev

## API Endpoints

GET /api/caterers

GET /api/caterers/:id

POST /api/caterers
